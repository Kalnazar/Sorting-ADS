# Sorting-ADS

### Description

In this project, you can see the 4 most common sorts, these are <b>bubble sort</b>, <b>heap sort</b>, <b>merge sort</b>, <b>quick sort</b>!

### Time complexity of sorts

#### Bubble Sort - O(n^{2})
#### Heap Sort - O(n\log n)
#### Merge Sort - O(n*log n)
#### Quick Sort - O(n*log n)
